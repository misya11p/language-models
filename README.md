## 言語モデルとは

単語の並びに確率を割り当てるモデル。

基本的には文章生成に使われるので、文章生成モデルとみてもいいかもしれない。文章生成では、与えられた文脈と語彙の下で次に続く単語を予測し、その単語を文脈に含めた上でまた次の単語を予測する。これを繰り返すことで文章を生成する。
